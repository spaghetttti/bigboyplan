# agentic_development

Working context for agentic development tasks on DevTrack (`tracker/`).

## Files

- `SPEC.md` — full project specification (schema, pages, components, design system)
- `architecture.md` — runtime shape, data layer patterns, auth flow, integration details
- `context.md` — quick onboarding summary for new agents
- `bugs.md` — open bugs and resolved fixes log
- `todo.md` — feature backlog and completed work
- `google-calendar-notion-plan.md` — implementation plan for Google Calendar + Notion integrations (not yet built)

## Source of truth

Primary implementation lives in `tracker/`. These docs are kept in sync manually — when in doubt, read the code.
